    Class ExprParser

    example of an expression parser. expr -> term { + term } term -> factor { * factor } factor -> number | ( expr )

    Example

    unittest for the expression parser 

    auto parser = new ExprParser;
auto p = parser.expr();
auto res = p.parse("1+2*3");
assert(res.success);
assert(res.results[0] == 7);
res = p.parse("(1+2)*3");
assert(res.success);
assert(res.results[0] == 9);
